# Opções de Tipo de Demanda para Consultas (CE/SE)

Ao preencher a coluna "Tipo de Demanda" (Coluna M) nas abas CE ou SE, utilize uma das seguintes opções:

1. **AESP - BD - Somente Consulta ou Consulta Prévia**
2. **AESP - Geração de Relatórios**
3. **AESP - Reexecução**
4. **AESP - Execução - Alteração de Parâmetros**

Estas opções foram identificadas a partir do arquivo opcoes_consulta.png.
